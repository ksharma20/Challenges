This is an assignment task for kinderPass

Following are the Apis :

  • Sign up
  • Login or Logout
  • Create Task
  • View Tasks
  • Mark Task Complete
    Use Endpoint --> /api/task/{id}
    And Send a POST request payload as 
    {"option":"complete"}
    OR 
    {"option":"mark"}
  • Delete Task
    Use Endpoint --> /api/task/{id}
    And Send a POST request payload as 
    {"option":"delete"}
    OR 
    {"option":"del"}
